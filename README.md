Repo for Configs, Errors & Rule sets.

Easier than mass-updating servers.

## File Types
Errors: These are errors that appear in the console. This will have a search key & a message that will be given  to the user via Builder Sherman.

Config Rules: Used by kpconfig. Useful for games that have Quotations, arrays that the config does not fully support.

Extra Configs: Handles diffcult config shit for eg palworld.
